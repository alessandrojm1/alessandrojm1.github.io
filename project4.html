3D Dice Visualization

An interactive visualization tool built with Python, PyGame, and OpenGL to simulate and explore polyhedral dice.

Overview

Developed an interactive Python program using PyGame and OpenGL to model and animate polyhedral dice in a dynamic environment. The program renders five distinct dice (tetrahedron, cube, octahedron, dodecahedron, and icosahedron) with smooth 3D- style rotations, resulting in a realistic simulation. Custom keyboard controls allow users to dynamically switch between dice types, making the tool both functional and engaging for demonstrations or exploration.

Tech Stack

Python | PyGame | OpenGL
